# BuildSensei

BuildSensei es un sistema experto diseñado para eliminar la incertidumbre al armar una computadora. Utilizando la lógica de sistemas expertos, garantiza que cada componente seleccionado sea 100% compatible y, lo más importante, esté óptimamente balanceado para el rendimiento deseado.

![Logo](frontend/img/Logo2.png)

## Características

*   **Selección de Componentes**: Elige entre una amplia variedad de componentes de hardware (CPUs, GPUs, etc.).
*   **Verificación de Compatibilidad**: El sistema experto valida que todos los componentes seleccionados sean compatibles entre sí.
*   **Optimización de Rendimiento**: Recibe recomendaciones para asegurar que tu configuración esté balanceada y ofrezca el mejor rendimiento.
*   **Interfaz Intuitiva**: Una interfaz de usuario amigable que te guía a través del proceso de construcción.

## Tecnologías Utilizadas

*   **Backend**: Python, Flask, SQLite
*   **Frontend**: HTML, CSS, JavaScript

## Cómo Empezar

Sigue estos pasos para ejecutar el proyecto en tu máquina local.

### Prerrequisitos

*   Python 3.13
*   Flask

### Instalación y Ejecución

1.  Clona el repositorio (o descarga el código fuente).

2.  Instala las dependencias necesarias. Por ahora, solo necesitas Flask:
    ```sh
    pip install Flask
    ```

3.  Ejecuta la aplicación desde el directorio raíz del proyecto:
    ```sh
    python backend/app.py
    ```

4.  Abre tu navegador y ve a `http://127.0.0.1:5000` para ver la aplicación en funcionamiento.
